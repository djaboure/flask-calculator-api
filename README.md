# Projet Flask : Système de Calcul API

## Contexte du projet
Ce projet est une application web Python basée sur le framework Flask. Elle inclut une interface d'accueil simple ainsi qu'une API de calcul fonctionnelle. 

L'équipe de développement a finalisé les fonctionnalités de base. L'application est stable et fonctionne parfaitement en environnement local. L'ensemble des fonctions et les scénarios de test ont été validés avec succès sur les postes de travail.

## État de l'art technique
- **Framework :** Flask
- **Tests :** Pytest (avec couverture de code)
- **Langage :** Python 3.9+

## Consigne
Votre mission est de mettre en place un pipeline CI (Intégration Continue) complet pour ce projet. Le pipeline devra automatiser les étapes suivantes en utilisant les outils mis à votre disposition :
1.  **Tests :** Exécution automatisée des scénarios de test.
2.  **Qualité :** Analyse statique du code via **SonarQube**.
3.  **Stockage :** Archivage des artefacts sur le serveur **Nexus**.

Les serveurs **SonarQube** et **Nexus** seront mis a votre disposition.
